Frequently Asked Questions

Get answers to common questions about our AI Prompt & Code Screenshot Generator.

What's the best size for social media sharing?

For Twitter/X, aim for 1200px width with optimal height. For LinkedIn, 1200-1600px width works well. Use the drag-to-resize handle to adjust dimensions that fit your platform's aspect ratio requirements.

How do the typography controls work?

Font size (14px-24px) controls text readability, while line height (1.4-2.0) adjusts spacing between lines. Larger font sizes work well for mobile sharing, while smaller sizes fit more content in the same space.

What makes this tool good for AI prompts?

The tool preserves original Markdown formatting while adding syntax highlighting, making complex prompts easy to read. Templates are optimized for social media, and the drag-to-resize feature ensures perfect dimensions for each platform.

What technology powers this tool?

We use Snapdom library for reliable DOM-to-image rendering, replacing complex Canvas-based systems. This ensures consistent quality across all browsers and eliminates previous rendering bugs.

Which template should I use?

Dark Code is our default (great for code), Modern Light for clean prompts, and Gradient for eye-catching social media posts. Each template is optimized for different content types and platforms.

How do social media presets work?

Select a platform from the Social Media Size dropdown to automatically set optimal width and height based on 2024-2025 official guidelines. Twitter/X uses 1280×720px (HD 16:9), Instagram Post is 1080×1080px (1:1 square), Instagram Story is 1080×1920px (9:16 vertical), and YouTube uses 1280×720px (HD thumbnail) for best results.

Is my data uploaded to any servers?

No, all processing happens entirely in your browser using Snapdom. Your AI prompts, code snippets, and generated screenshots never leave your device, ensuring complete privacy and security.
